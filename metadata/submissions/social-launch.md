## Developer Community Post

```text
If you are experimenting with AI visibility, GEO, AEO, or llms.txt, we open-sourced the MaxAEO AI Visibility MCP workflow.

It is designed for Claude, Codex, Cursor, and MCP-compatible agents:

- check /llms.txt
- check AI crawler access
- check robots and sitemap
- check canonical, noindex, metadata, and JSON-LD
- return a local-only AI visibility confidence score
- return a technical foundation score
- generate a 7-day action plan

Install:
npx -y maxaeo-ai-visibility-mcp

The free tool is local-first and public-web-only. It does not call MaxAEO APIs, LLM APIs, or hidden telemetry by default.
```
