# AI Crawler User Agents

Track crawler rules separately because search, training, and user-triggered browsing can use different agents.

Initial crawler set:

- GPTBot
- ChatGPT-User
- OAI-SearchBot
- ClaudeBot
- PerplexityBot
- Google-Extended
- Applebot
- Bingbot
- Baiduspider

Important caveat: allowing or blocking one crawler does not guarantee AI search recommendations. Treat this as crawlability readiness, not ranking proof.
