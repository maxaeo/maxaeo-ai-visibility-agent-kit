# 7-Day / 30-Day Action Plan

## 7 Days

1. Fix robots rules that block key AI crawler access.
2. Add or clean up `/llms.txt`.
3. Align `llms.txt` URLs with sitemap URLs.
4. Add title, description, canonical, and JSON-LD to the homepage.
5. Create answer-first sections on the top commercial pages.

## 30 Days

1. Build recurring buyer-intent prompt sets.
2. Add schema coverage for Organization, SoftwareApplication, Article, BreadcrumbList, and FAQ where supported by visible content.
3. Create comparison and alternatives pages for high-intent competitor queries.
4. Add sourceable statistics, examples, and author/entity signals.
5. Move from one-time checks to continuous MaxAEO monitoring.
